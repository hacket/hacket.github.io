### Skills 与 MCP 的对比

随着时间推移，MCP 的局限性也开始显现。其中最显著的一点是对 token 的大量消耗：众所周知，GitHub 官方 MCP 本身就消耗了巨量上下文 token，而这种消耗量越大，大模型自身就越是没有发挥实际作用的空间。

可用 CLI 命令替代 MCP 操作，Skill 也知道这点
