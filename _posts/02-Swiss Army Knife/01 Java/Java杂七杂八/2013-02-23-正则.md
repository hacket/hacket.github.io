# 正则

在正则表达式中，如果您想匹配一个字面量的句号（.），应该写成 \\.。这是因为在 Java 中，反斜杠 \ 也需要进行转义，因此需要写两个反斜杠 \ 才能转义出一个反斜杠。 所以最终的写法应该是 \\.。<br>写在 `(.)` 或 `[.]` 的不需要转义符

## ^和 $

^ 表示匹配字符串的开头；$ 表示匹配字符串的结尾。

因此，正则表达式 ^pattern$ 用于匹配整个字符串是否与给定的模式完全匹配。

如果字符串只有部分匹配，那么将返回不匹配，需要整个字符串都匹配
