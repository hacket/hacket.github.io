# 联想小新放静电

## 异常情况

1. 蓝牙键盘和蓝牙鼠标都用不了，删除设备重新连接也用不了

> 解决：放静电

## 放静电步骤

1. 关机
2. 长按开机键 20 秒以上
3. 松开开机键
