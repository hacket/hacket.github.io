## 快捷键冲突 ShortcutDetective

问题：在 VSCode 中的 `Ctrl+Shift+P` 失效了

解决：

`ShortcutDetective` 软件可以检测冲突按键。

<https://www.irradiatedsoftware.com/downloads/>

需要开启 Accessibility 权限：

`Open System Settings > Privacy & Security > Accessibility`
