# git 常用命令别名

## oh-my-zsh 中的 git 别名

oh my zsh 会自动启用 git 插件，默认有很多方便的别名，在此记录一下，可以使用下面的命令查看 git 插件具体替换了什么

```shell
cat ~/.oh-my-zsh/plugins/git/git.plugin.zsh
```

### 常用别名

查看别名：`which gcl`
